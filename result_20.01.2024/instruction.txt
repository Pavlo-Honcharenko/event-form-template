Содержимое файла third-JS.txt надо вставить самым последним скриптом в HTML, перед </body>, то есть после тех JS, которые я давал раньше.


Для того, чтобы расположить кнопу слева, надо оболочке кнопки добавить класс "leftward". Например: 
<div id="export_button" class="leftward" style="float: right; position: sticky; right: 5px; bottom: 0; margin-top: 5px;">
	<button class="active" onclick="exportRegistrations();">
		Export
	</button>&nbsp;&nbsp;
</div> 
В данном случае div id="export_button" является оболочкой для кнопки <button class="active"...> 
В коде есть кнопки без оболочки, в таком случае класс надо дать непосредственно этой кнопке. Класс "leftward" имеет только одно свойство float: left; .


Для информационного блока, который должен появляться под табами в Event Console Layer с текстом "Congrats, you event has been created!..." я сделал  id #ec_success_div и id #ec_success_content, они похожи на #ec_status_div и #ec_status_content, там цвет текста и иконка отличаются. 

Вот HTML код того элемента, который раньше был жёлтым:
<div id="ec_status_div" style="margin-top: 8px; display: block">
	<span style="display: block; float: right" ;=""><a style="text-decoration: none" onclick="hideStatus();">[X]</a>&nbsp;</span>
	<div id="ec_status_content" style="font-size: 13px">
		1 registration updated successfully.
	</div>
</div>
